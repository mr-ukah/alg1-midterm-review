Q8
Linear Equations · Point on a Line
The graph of a linear equation contains the points (3, 11) and (−2, 1). Which point also lies on the graph?
(a) (2, 1) (b) (2, 4) (c) (2, 6) (d) (2, 9) ✓
Plan of Attack
Strategy: find the equation of the line first

Two points determine a line. Use the two given points to find the slope, then write the equation. Once you have the equation, plug in x = 2 and see which y-value you get.

Why all answer choices use x = 2

The question is really asking: what is f(2)? All four choices give different y-values for x = 2. Find the right one by computing.

⚠️ Don't guess from the pattern. The points (3, 11) and (−2, 1) might look like "+1 per unit" but the slope is actually 2, not 1. Always calculate.
↗ Study Guide: Linear Models
Full Solution
1Find the slope using the two given points
m = (11 − 1) / (3 − (−2)) = 10 / 5 = 2
2Write the equation using point-slope form
Use (3, 11) and slope m = 2: y − 11 = 2(x − 3) y = 2x − 6 + 11 = 2x + 5
3Find f(2)
y = 2(2) + 5 = 4 + 5 = 9 The point (2, 9) lies on the graph. ✓
✗Why the wrong choices are wrong
(a) (2, 1): y = 1 would need 2x + 5 = 1 → x = −2. That's a different point already given. (b) (2, 4): confuses 2(2) = 4 without adding b. Forgot the +5. (c) (2, 6): might come from thinking slope is 1 (not 2): 1(2) + 5 − 1 = 6. Wrong slope.
✓
Answer (d) (2, 9) The equation of the line is y = 2x + 5. At x = 2, y = 9.
Q1
Quadratic Functions · Minimum Value · Table vs. Equation
The quadratic functions r(x) and p(x) are given below. The function with the smaller minimum value is:
p(x) = x² + 2x − 8
(a) p(x) and the value is −9 (b) p(x) and the value is −1 (c) r(x) and the value is −16 ✓ (d) r(x) and the value is −2
Plan of Attack
Two different sources, same goal

r(x) comes from a table — its minimum is just the smallest number in the r(x) column. p(x) comes from an equation — its minimum is the y-value at the vertex, found using x = −b/(2a).

Compare the minimum VALUES — not the x-values

The question asks which function's minimum is smaller. Once you find both minimums, just compare the two numbers.

⚠️ Don't confuse the x-value of the vertex with the minimum value. The vertex of p(x) is at x = −1, but the minimum VALUE is what you get when you plug x = −1 back in. The x tells you where; the y tells you the actual minimum.
↗ Study Guide: Quadratic Models
Full Solution
1Find the minimum of r(x) from the table
r(x) values: −12, −15, −16, −15, −12, 7 Smallest value: −16, occurring at x = −2 Minimum of r(x) = −16
2Find the minimum of p(x) = x² + 2x − 8
a = 1, b = 2, c = −8. Since a > 0, parabola opens up → has a minimum. x at vertex: x = −b/(2a) = −2/(2·1) = −1 Minimum value: p(−1) = (−1)² + 2(−1) − 8 = 1 − 2 − 8 = −9 Minimum of p(x) = −9
3Compare: which is smaller?
r(x) minimum: −16    p(x) minimum: −9 −16 < −9, so r(x) has the smaller minimum value.
✗Why the wrong choices are wrong
(a) p(x) and −9: p(x) has minimum −9, but r(x)'s minimum (−16) is smaller. Wrong function chosen. (b) p(x) and −1: −1 is the x-value of p(x)'s vertex, not the minimum value. Confused x and y. (d) r(x) and −2: −2 is the x-value where r(x) is minimized, not the minimum value. Same mistake.
✓
Answer (c) r(x) and the value is −16 r(x) minimum = −16 (from table). p(x) minimum = −9 (from vertex formula). −16 < −9.
Q7
Exponent Rules · Rewriting Expressions
If x = 8^a, then the value of 8x is
(a) x + 1 (b) a + 8 (c) 8^(a+1) ✓ (d) 9a
Plan of Attack
Strategy: substitute, then apply the exponent rule

Replace x with 8^a. Then you have 8 · 8^a. Both factors have the same base (8), so you add the exponents. The plain 8 is really 8¹.

Rule: bᵐ · bⁿ = b^(m+n)

When multiplying powers with the same base, add the exponents. Don't multiply the bases — the base stays the same.

⚠️ 8 · 8^a is NOT 64^a and NOT 8^(8a). You're multiplying (adding exponents), not raising to a power (which would multiply exponents). The base stays 8.
↗ Study Guide: Exponent Rules
Full Solution
1Substitute x = 8^a
8x = 8 · 8^a
2Rewrite 8 as 8¹ and add exponents
= 8¹ · 8^a = 8^(1+a) = 8^(a+1)
✗Why the wrong choices are wrong
(a) x + 1: adding 1 to x, not 1 to the exponent. Confuses arithmetic addition with exponent addition. (b) a + 8: adds the exponent a to the base 8. These are different things — you can't add them. (d) 9a: multiplies 8+1=9 by a. Confused "8 times" with "8 plus 1" and then multiplied the exponent instead of adding.
✓
Answer (c) 8^(a+1) 8 · 8^a = 8¹ · 8^a = 8^(1+a). Same base, add the exponents.
Q11
Functions · Domain · Discrete vs. Continuous
At an ice cream shop, the profit P(c) is modeled by P(c) = 0.87c, where c represents the number of ice cream cones sold. An appropriate domain for this function is
(a) an integer ≤ 0 (b) an integer ≥ 0 ✓ (c) a rational number ≤ 0 (d) a rational number ≥ 0
Plan of Attack
Two decisions: integer or rational? And ≥ 0 or ≤ 0?

c = number of cones sold. Ask yourself: (1) Can you sell half a cone? No → must be an integer. (2) Can you sell a negative number of cones? No → must be ≥ 0.

Discrete vs. continuous

Cones sold is a counting number — you count them as whole numbers. It's a discrete quantity. Rational numbers include fractions (like 2.5 cones), which don't make sense here.

⚠️ 0 is included. The shop could sell zero cones (a bad day). So the domain includes 0, which means ≥ 0, not > 0.
↗ Study Guide: Functions & Domain
Full Solution
1Integer or rational?
c = number of cones sold. You can't sell 2.5 cones — cones come in whole units. → c must be an integer (whole number).
2≥ 0 or ≤ 0?
You can't sell a negative number of cones. Minimum possible sales = 0 (sell nothing). → c ≥ 0
✗Why the wrong choices are wrong
(a) integer ≤ 0: correct type (integer) but wrong direction. Negative cones don't exist. (c) rational ≤ 0: wrong type AND wrong direction. You can't sell 1.7 cones, and you can't sell negative cones. (d) rational ≥ 0: right direction but wrong type. Half a cone isn't a real transaction — cones are discrete whole units.
✓
Answer (b) an integer ≥ 0 In set notation: {c | c ∈ ℤ, c ≥ 0}. Cones are whole units; you can't sell a negative or fractional cone.

Q15
Sequences · Exponential Behavior
Determine and state whether the sequence 1, 3, 9, 27, ... displays exponential behavior. Explain how you arrived at your decision.
Plan of Attack
Two-step test: arithmetic first, then exponential

Step 1: Check if the differences are constant (that would make it arithmetic). Step 2: Check if the ratios are constant (that would make it exponential). A sequence is exponential if — and only if — the ratio between consecutive terms is always the same non-zero number.

What the explanation needs to say

You must state the test you applied AND show the calculation. Just saying "yes it's exponential" gets 0 credit. Show the ratios.

↗ Study Guide: Sequences
Full Solution
1Test for arithmetic: are the differences constant?
3 − 1 = 2    9 − 3 = 6    27 − 9 = 18 Differences: 2, 6, 18 — NOT constant. Not arithmetic.
2Test for exponential: are the ratios constant?
3 ÷ 1 = 3    9 ÷ 3 = 3    27 ÷ 9 = 3 Ratios: 3, 3, 3 — constant. Common ratio = 3. ✓
3State the conclusion
Yes, the sequence displays exponential behavior. The ratio between each consecutive pair of terms is constantly 3. Each term is multiplied by 3 to get the next term. This is the definition of an exponential sequence.
✓
Full Credit Answer Yes — common ratio = 3 Must show: ratios calculated (3/1, 9/3, 27/9 all = 3) AND state they are constant AND conclude exponential. Both points require the explanation to be present.
Q13
Literal Equations · Solving for a Variable
The formula for converting degrees Fahrenheit (F) to degrees Kelvin (K) is: K = (5/9)(F + 459.67). Solve for F, in terms of K.
Plan of Attack
Undo in reverse order of operations

To get from F to K, the formula: (1) adds 459.67 to F, then (2) multiplies by 5/9. To reverse this and isolate F, you undo in the opposite order: (1) multiply both sides by 9/5, then (2) subtract 459.67.

Treat K exactly like a number

K is just a placeholder for "whatever the Kelvin value happens to be." Do algebra on it exactly as you would with any number.

⚠️ Multiply by 9/5, not 5/9. You're undoing multiplication by 5/9, so you multiply by its reciprocal: 9/5.
↗ Study Guide: Literal Equations
Full Solution
1Start with the formula
K = (5/9)(F + 459.67)
2Multiply both sides by 9/5
(9/5)K = F + 459.67
3Subtract 459.67 from both sides
F = (9/5)K − 459.67
✓Quick check with a known value
Water freezes at 0°C = 32°F = 273.15 K F = (9/5)(273.15) − 459.67 = 491.67 − 459.67 = 32°F ✓
✓
Answer F = (9/5)K − 459.67 Multiply both sides by 9/5, then subtract 459.67. Both steps must be shown for full credit.
